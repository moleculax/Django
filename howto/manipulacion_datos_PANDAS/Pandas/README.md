# 🐼 HowTo Pandas

## 📌 Introducción
[Pandas](ca://s?q=Que_es_pandas) es una librería de Python para **análisis y manipulación de datos**.  
Permite trabajar con estructuras como **Series** y **DataFrame**, ideales para datos tabulares y series temporales.

---

## 🔧 Instalación
```bash
pip install pandas
```
