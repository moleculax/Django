---

## 📊 ¿Qué es Pandas?

**Pandas** es una librería de Python para **análisis y manipulación de datos**. Proporciona estructuras de datos flexibles y eficientes como:

| **Estructura** | **Descripción** |
|:---|:---|
| `Series` | Array unidimensional etiquetado |
| `DataFrame` | Tabla bidimensional con columnas y filas |
| `Index` | Etiquetas para filas y columnas |

## 🔗 Integración Django + Pandas

### ¿Para qué sirve?

| **Caso de uso** | **Ejemplo** |
|:---|:---|
| 📊 **Reportes** | Generar reportes desde modelos Django |
| 📈 **Dashboards** | Crear dashboards con datos agregados |
| 🔍 **Análisis** | Analizar datos de usuarios, ventas, etc. |
| 📁 **Importación/Exportación** | Leer/Guardar CSV, Excel, JSON |
| 🧹 **Limpieza de datos** | Limpiar y transformar datos antes de guardar |

## 🛠️ Instalación

### En entorno virtual (recomendado)

```bash
# Crear y activar entorno virtual
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# Instalar Django y Pandas
pip install django pandas
```
